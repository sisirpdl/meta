This section contains information about developing WebXR experiences for Browser.

WebXR Device API lets you develop immersive VR experiences on the web, and is supported in multiple browsers. For more information, visit the WebXR home page: [https://immersiveweb.dev/](https://immersiveweb.dev/)

The WebXR explainer provides some background and technical information about the design and usage of the WebXR Device API. [https://immersive-web.github.io/webxr/explainer.html](https://immersive-web.github.io/webxr/explainer.html)
